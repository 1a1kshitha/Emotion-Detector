Emotion Detection Web Application
Overview

This project implements a simple AI-powered emotion detection system that analyzes the emotional tone of a given text. The system sends the input text to an NLP emotion prediction service and returns scores for different emotions expressed in the text.

The application is packaged as a Python module and deployed using a Flask web server so that users can test emotion predictions through a browser.

Key Capabilities

Accepts text input from a user

Sends the input to an emotion analysis API

Retrieves probability scores for different emotions

Identifies the dominant emotion

Handles invalid or empty inputs gracefully

Provides a simple web endpoint for testing

Technologies Used

Python

Flask (for web deployment)

Requests library (for API communication)

Git and GitHub for version control
